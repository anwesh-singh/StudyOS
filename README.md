# 📖 About

**StudyOS** is a modern AI-powered learning platform built to help students study more effectively from a single dashboard.

Instead of switching between multiple apps, StudyOS brings notes, planning, analytics, and AI-powered learning into one unified workspace.

The long-term goal is to build a complete AI learning ecosystem for students.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/anwesh-singh/StudyOS.git
```

Go to the project folder:

```bash
cd StudyOS/frontend
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

# 🔑 Environment Variables

Create a `.env.local` file inside the `frontend` folder.

```env
OPENAI_API_KEY=your_openai_api_key
```
